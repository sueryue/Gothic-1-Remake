## 🎨 截图技巧

### **最佳拍摄设置:**

```bash
✅ 关闭HUD (hide health bar/minimap for cinematic shots)
✅ 使用第三人称视角 (third-person camera angle)
✅ 调整焦距到主体 (focus on main subject)
✅ 避免UI遮挡 (no UI elements if possible)
✅ 选择黄金时刻 (golden hour lighting)
✅ 捕捉动态瞬间 (action shots - combat, spells)
```

### **不推荐的截图:**

```bash
❌ 黑屏/加载界面 (black screen/loading)
❌ 模糊/掉帧 (frozen/framerate drop)
❌ 重复内容 (duplicate screenshots)
❌ 纯文本菜单 (pure text menus - 用截图+标注即可)
```

## 🖼️ 图片优化标准

### **格式要求:**

```yaml
Web格式:
  - 主图: WebP (比JPEG小30%,质量相当)
  - 动图: GIF/WebP (用于展示动态效果)
  - 图标: SVG (如果可能)

压缩标准:
  - 最大宽度: 1920px (桌面端)
  - 移动端宽度: 800px
  - 文件大小: < 500KB/张
  - 质量: 80-85% (平衡画质和加载速度)
```

### **命名规范:**

```bash
✅ 小写字母 + 连字符 (kebab-case)
✅ 描述性文件名 (英文)
✅ 标注版本号 (如果有多个版本)

例:
  ✅ lock-5plate-step1.jpg
  ✅ old-camp-fire-night.jpg
  ❌ ScreenShot001.jpg
  ❌ 开锁步骤1.jpg
```

## 🔍 带标注的截图 (Annotation)

### **使用工具:**

```
推荐:
  ✅ Paint.NET (免费, Windows)
  ✅ GIMP (免费, 跨平台)
  ✅ Photoshop (付费, 专业)
  ✅ Figma (在线协作)
```

### **标注类型:**

```markdown
1️⃣ 箭头指示 (Arrow Pointers)
   - 红色箭头指向关键物体
   - 黄色高亮按钮位置

2️⃣ 数字编号 (Numbered Steps)
   - 圆形数字徽章 (1️⃣2️⃣3️⃣)
   - 对应步骤说明

3️⃣ 圈注 highlight
   - 蓝色圆圈强调区域
   - 半透明背景突出显示

4️⃣ 文字叠加 (Text Overlay)
   - 白色字体+黑色描边
   - 简短说明 (<10 words)
```

### **标注示例:**

```
[开锁截图]
→ 红色箭头指向"蓝色按钮"
→ 数字①标注"第一步"
→ 底部文字:"Press Blue First!"

[派系对比截图]
→ 三个并排营地照片
→ 每张下方标注名称+颜色代码
→ 箭头指向关键NPC
```

## 🚀 快速开始步骤

### **Step 1: 启动游戏录制**
```bash
推荐工具:
  ✅ OBS Studio (免费, 直播+录制)
  ✅ NVIDIA Share (GeForce Experience)
  ✅ Xbox Game Bar (Windows Built-in)
```

### **Step 2: 按清单截图**
```
按照上面的"必备截图清单"逐一拍摄
每张照片确保:
  - 清晰度 > 1080p
  - 光线充足 (不用夜视模式)
  - 无UI遮挡 (或特意展示UI)
```

### **Step 3: 整理+重命名**
```bash
# PowerShell批量重命名示例
Get-ChildItem "C:\Screenshots\*" -Filter "*.jpg" |
  Select-Object -First 20 |
  ForEach-Object {
    $newName = "lock-5plate-" + $_.Name;
    Rename-Item $_ $newName
  }
```

### **Step 4: 上传到网站**
```bash
拖放到 assets/screenshots/ 目录
提交Git:
  git add assets/screenshots/*
  git commit -m "Add gameplay screenshots"
```
