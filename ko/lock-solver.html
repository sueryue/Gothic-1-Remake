✅ Your Complete Solution Sequence

📖 Understanding the Algorithm

The Gothic 1 Remake lockpicking puzzle is a constraint satisfaction problem. Here's how it works:

🎯 Goal State

All plates must reach position 0 (center) simultaneously. The challenge: plates are mechanically linked!

🔧 Constraints

Each plate pair may be connected via gears: Same Direction (both move together) or Opposite (reverse). These connections create cascading effects.

💡 Solution Strategy

Our solver uses a breadth-first search (BFS) algorithm to find the minimum number of moves from current state to goal state, respecting all linkage constraints.
