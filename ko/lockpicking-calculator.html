Understanding Button Colors

Color Effect Difficulty Impact
Red Changes ONLY this button Easiest to predict
Blue Changes this button + adjacent buttons Moderate complexity
Yellow Changes this button + entire row/column Hard to predict
Empty No button present Skip this position

Key Rules

  • Goal: Make ALL buttons turn green
  • Caution: Wrong attempts damage the lock
  • Limit: Too much damage = broken lock (unopenable!)
  • Strategy: Always find the correct sequence first
