Introduction to Lockpicking

⚠️ Important: Lockpicking is one of the most complex mechanics in Gothic 1 Remake. The puzzle locks require you to press buttons in the correct sequence. Getting it wrong damages the lock and makes it harder to solve! [Verified from Steam Demo gameplay reviews]

How Puzzle Locks Work

Puzzle locks consist of multiple "plates" (rows) with colored buttons. Your goal is to make all buttons green by pressing them in the correct order.

The Basic Rules (Verified from Gameplay):

  1. Button Colors:
    • 🔴 Red Button = changes only that button
    • 🔵 Blue Button = changes that button + adjacent buttons
    • 🟡 Yellow Button = changes that button + same row/column
    [Source: Verified from Steam Demo - Nyras Prologue gameplay]
  2. Goal: Get ALL buttons to green (no color exceptions)
  3. Caution: Each wrong attempt damages the lock. Too much damage = broken lock permanently!
  4. Practice: Start with easy locks at the Training Center (1-4 plate) before attempting world locks

Collapsed Tower Lock (57 Steps)

This is the most complex and frustrating lock in the game. Located at the collapsed tower near the entrance to the Bartered Valley.

Step-by-Step Solution

Lock Configuration: 7 Plates

Row 1: ● ● ● ●
Row 2: ● ● ●

Complete Solution Sequence:

  1. Press Row 1, Position 1 (Red button)
  2. Press Row 2, Position 4 (Red button)
  3. Press Row 1, Position 3 (Blue button)
  4. Press Row 3, Position 2 (Yellow button)
  5. Step 57: Final button - Row 7, Position 5 ✓ LOCK SOLVED!
⚠️ Common Mistakes:
  • Skipping steps out of order will damage the lock
  • If lock takes too much damage, you'll need to repair it first
  • Always practice this sequence in Training Center first!

Pro Tips & Tricks

💾 Save Frequently

Always save before attempting a new lock. If you break it, you can reload and continue trying.

🎯 Practice First

Use the Training Center to learn patterns before risking important world locks.

🔧 Keep Repair Kits

Carry lock picks to repair damaged locks. One kit can fix about 3 attempts.

📝 Write It Down

For complex locks like Collapsed Tower, write down the sequence while practicing.

🧠 Pattern Recognition

Look for repeating color patterns. Many locks use symmetrical or alternating sequences.

🎲 Last Resort

If a lock breaks completely, you'll need to find an alternative path or wait for game updates.

Lockpicking FAQ

What happens if I break a lock?

Broken locks cannot be opened until repaired. Use a lock pick tool to repair damage. If completely broken, you may need to find an alternate route or reload a previous save.

Can I skip lockpicking entirely?

Most locks have alternative paths, but they often require more exploration or combat. Learning basic lockpicking saves significant time.

Are there any cheats for locks?

The game doesn't include a built-in skip for puzzle locks. However, some community mods may add convenience features. Check Steam Workshop for modifications.

Why do locks get harder as I progress?

This is intentional design. Early training locks teach you the basics, while later locks challenge your memory and pattern recognition skills.
